# 🤖 Maqueen Lite Field Guide — Android wrapper

Capacitor wrapper that packages the [maqueen-activities](https://github.com/abourdim/maqueen-activities) web app as an installable **Android APK**.

> دليل ماكوين لايت الميداني · Maqueen Lite Field Guide

A complete field guide for the DFRobot Maqueen Lite (ROB0148) micro:bit robot: 85 activities across nine categories from first motion to autonomous swarm and Q-learning. Six learning paths, copy-paste code in JavaScript and Python, full trilingual EN/FR/AR with automatic RTL Arabic layout.

## Quick start

```bash
./manage.sh    # auto-detects your OS, then opens the menu
```

Build, test on emulator, and prepare a Play Store release from a single guided menu. See [WORKSHOP.html](WORKSHOP.html) for the full step-by-step guide.

## Download the latest APK

[📦 v1.0-debug release](https://github.com/abourdim/maqueen-activities-android/releases/latest) · [![QR](releases/download-qr.svg)](https://github.com/abourdim/maqueen-activities-android/releases/latest)

Scan the QR with an Android phone, or `adb install -r maqueen-activities-v1.0-debug.apk`.

## Capabilities required at runtime

- Bluetooth Low Energy (Web Bluetooth → @capacitor-community/bluetooth-le)
